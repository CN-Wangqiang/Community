### 社区介绍
学习SpringBoot的入门实践，[ElasSearch中文社区](https://elasticsearch.cn/)的风格。

### 技术工具
项目采用SpringBoot构建。前端使用Thymeleaf模板引擎页面。数据库为MySQL。
